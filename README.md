<details>
  <summary><h2>⚙️ Running locally for development</h2></summary>

Start cloning Bobify:

```
git clone https://github.com/staxhinho/Bobify.git
```


Next, install its dependencies (make sure npm is installed on your machine):

```
npm install
```

To start, run:

```
npm start
```
